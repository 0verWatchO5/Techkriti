## üìä Dashboard Tour

### The Dashboard (`dashboard.html`)
Your business at a glance:
- **Total Revenue** - Sum of all sales
- **Total Expenses** - Sum of all expenses
- **Inventory Value** - Total inventory worth at cost
- **Net Profit** - Revenue minus expenses

Charts show:
- Profit trends over time
- Expense breakdown by category
- Top-selling products

### Managing Expenses (`expenses.html`)
1. Click "+ Add Expense" button
2. Fill in category, amount, date
3. Optionally add description
4. Mark as paid or pending
5. View all expenses in the table below

**Features:**
- Filter by category and status
- Search expenses
- View monthly trends
- Pie chart of expense categories

### Managing Inventory (`inventory.html`)
1. Click "+ Add Product" button
2. Enter product details (name, SKU, category, cost, price)
3. Set current quantity and minimum stock level
4. System tracks inventory value automatically

**Features:**
- Low stock warnings
- Inventory value calculations
- Stock status visualization
- Edit/delete products

### Recording Sales (`sales.html`)
1. Click "+ Add Sale" button
2. Select product from dropdown
3. Enter quantity (system checks availability)
4. Inventory is automatically deducted
5. Record customer name and date

**Features:**
- Auto-inventory deduction
- Sales trends and top products
- Monthly sales metrics
- Order value calculations

### Analytics & Profits (`analytics.html`)
Deep dive into your business performance:
- Net profit and profit margin
- Revenue vs expenses comparison
- Growth metrics (month-over-month)
- Top 5 products and expense categories
- Financial summary dashboard

## üéØ Common Tasks

### Task 1: Add an Expense
```
1. Go to Expenses page
2. Click "+ Add Expense"
3. Category: Select one
4. Amount: Enter dollar amount
5. Date: Pick a date
6. Status: Paid or Pending
7. Click "Save Expense"
```

### Task 2: Add a Product to Inventory
```
1. Go to Inventory page
2. Click "+ Add Product"
3. Product Name: Give it a name
4. SKU: Unique code (e.g., PROD-001)
5. Category: Choose category
6. Unit Cost: Cost per item
7. Unit Price: Selling price
8. Current Quantity: How many in stock
9. Minimum Stock: Alert when below this
10. Click "Save Product"
```

### Task 3: Record a Sale
```
1. Go to Sales page
2. Click "+ Add Sale"
3. Product: Choose from dropdown
4. Quantity: Enter quantity to sell
5. Unit Price: Auto-filled from inventory
6. Customer Name: Enter customer name
7. Date: Sale date
8. Status: Completed or Pending
9. Click "Save Sale"
‚Üí Inventory automatically reduces by quantity sold!
```

### Task 4: Check Profitability
```
1. Go to Analytics page
2. View Net Profit card at top
3. Check Profit Margin %
4. Review Month-over-Month growth
5. Analyze Revenue vs Expenses chart
6. Check top products and expense categories
```

## üíæ Your Data

**Where is my data stored?**
- All data is stored in your browser's localStorage
- It's secure and private
- No data leaves your computer
- Data persists even after closing the browser

**Can I backup my data?**
- Currently stored locally only
- Data is not synced to cloud
- To backup: Use browser export or save locally

**What if I clear my browser data?**
- All BizPulse data will be deleted
- Be careful when clearing browser cache!

## ‚ùì FAQ

**Q: Do I need internet?**
A: No! The app works completely offline. No internet connection required.

**Q: Is my data safe?**
A: Yes! All data stays on your computer in browser storage. Nothing is uploaded.

A: Clear your browser's localStorage or use Developer Tools to manually clear it.
